# AI_Edu_HW2: Использование ИИ для анализа приложений и быстрого онбординга

## Описание домашнего задания

### Тема
Использование искусственного интеллекта для анализа приложений и обеспечения быстрого heboarding новых разработчиков.

### Цели задания
1. Изучить возможности современных ИИ-инструментов в контексте анализа программного обеспечения
2. Разработать эффективные практики быстрого онбординга с использованием AI-ассистентов
3. Применить полученные знания на практике для анализа реальных проектов

### Задачи

#### Задание 1: Анализ архитектуры приложения
- Изучите исходный код проекта deepeval (находится в текущей директории)
- Используя ИИ-инструменты, проведите анализ архитектуры проекта
- Выявите ключевые компоненты, зависимости и структуру кода
- Создайте документацию по результатам анализа

#### Задание 2: Создание AI-ассистента для онбординга
- Разработайте инструкции по использованию ИИ для нового разработчика
- Подберите наиболее эффективные инструменты для работы с кодом проекта
- Создайте чек-лист для быстрого старта в проекте

#### Задание 3: Рефакторинг и улучшения
- Используйте ИИ для предложения улучшений существующего кода
- Примените рекомендации для оптимизации проектных файлов
- Документируйте изменения и их влияние

### Технологии и инструменты
- Python
- Git
- ИИ-ассистенты (GitHub Copilot, Cursor, или другие)\
- IDE: PyCharm

### Критерии успешного выполнения
- Качественный анализ архитектуры приложения
- Рабочий чек-лист онбординга
- Визуальное подтверждение результатов (скриншоты, диаграммы)
- Полная документация процесса
