## 🚀 Краткое описание исследуемого приложения

**DeepEval** — это фреймворк для автоматизированного тестирования и оценки больших языковых моделей (LLM) и мультимодальных моделей (MLLM). Система предоставляет инструменты для:

- Валидации качества ответов (концептуальная правильность, токсичность, PII)
- Обнаружения галлюцинаций
- Оценки релевантности и контекстуальной точности (RAGAS)
- Анализа настроения и биасов в генерации текста
- запуска тестовых случаев для CI/CD

### Ключевые возможности системы

1. **Тестирование LLM-моделей** — validation on соответствие ожиданиям
2. **Оценка чат-ботов** — проверка сохранения контекста между запросами
3. **Сравнение моделей (Arena)** — парное сравнение двух или более LLM
4. **RAG-системы** — оценка контекстуальной точности и полноты
5. **Коллекция бенчмарков** — MMLU, GSM8K, HumanEval, BBH и другие
6. **Интеграция с ML-инструментами** — CrewAI, LangChain, LlamaIndex, Pydantic AI
7. **Веб-интерфейс** — визуализация метрик и экспорт результатов (local server)

### Архитектурные компоненты

- **Dataset Service** — импорт и управление тестовыми данными
- **Test Case Service** — создание тестовых сценариев (LLMTestCase, ConversationalTestCase)
- **Evaluation Engine** — оркестрация выполнения метрик
- **Metrics Service** — вычисление показателей (HallucinationScore, RelevancyScore, Toxicity и др.)
- **Model Adapter** — абстракция LLM/Embedding моделей (OpenAI, HuggingFace, Ollama)
- **Benchmark Service** — загрузка стандартных наборов данных
- **CLI Gateway** — командная строка и локальный сервер результатов
